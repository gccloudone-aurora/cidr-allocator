## Requirements

Kubernetes: `>= 1.16.0-0`

## Installation

Install using Helm

```bash
helm repo add statcan-ca https://statcan.github.io/cidr-allocator
helm repo update
helm install cidr-allocator statcan-ca/cidr-allocator
```
